CompressXcodeImageAssets
========================

A script to compress images contained in Xcode Images.xcassets folders using pngquant

###Installation
- Download [pngquant](http://pngquant.org/pngquant.tar.bz2 "Mac Binary") and install to a exported path (Should be in your $PATH varible or somewhere like /local/bin or /usr/bin)

###Usage
```bash
$ ./compressImages.sh Avocado-iOS/Images.xcassets
```
